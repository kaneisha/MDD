Celiac Tracker is an application that lets you check the ingredients in a food item. It's main purpose is to make it easy for those with Celiac Disease see what they can and cannot eat while they're on the go. The app was made for a project in my Deployment of Flash Projects class. The API used for this project was the Nutrionix API.
